바쿠스 비밀번호 관리 시스템
--------
```bash
npm install

npm test        # Static type check (flow)
npm run build   # Build everything at once
open index.html

npm start       # Do everything excep flow in watch mode
                # Go to http://localhost:8080/webpack-dev-server/
```
